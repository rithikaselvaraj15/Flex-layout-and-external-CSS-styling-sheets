Welcome

I created this web page using HTML5 and CSS to build a clean, responsive layout. The structure includes a header with a logo and navigation links, a sidebar with collapsible menus for Dashboard, Reports, and Settings, and a main content area for the welcome message. I used Flexbox to align elements, making the sidebar and main content appear side by side while centering the welcome block both vertically and horizontally. The sidebar features background colors, hover effects, and rounded menus, while the main content has a white background for emphasis. I also applied a full-page background color using a linear gradient, giving a continuous colored block from top to bottom. The design is responsive, avoids fixed heights, and uses padding and max-width to ensure readability, creating a professional and user-friendly layout.

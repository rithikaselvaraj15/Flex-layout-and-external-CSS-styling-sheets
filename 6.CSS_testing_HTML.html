CSS EXTERNAL STYLES-SHEET

What is CSS External stylesheet

An External style sheet in CSS is a separate.css file where you write all your CSS rules and then link it to your HTML file. This keeps your HTML clean and makes it easier to style multiple pages at once.

  • Instead of writing styles inside your HTML, you write them in a different file.
  • You link that file to your HTML using the link tag in the head section.
  • If you want to change the design of multiple pages, you only need to edit the CSS file once.
